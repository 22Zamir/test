# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
SECRET_KEY=django-insecure-your-secret-key-here
KEITARO_API_URL=https://your-keitaro-instance.com
KEITARO_API_KEY=your-api-key-from-keitaro

# Опциональные настройки (можно оставить пустыми)
KEITARO_DOMAIN=
KEITARO_GROUP=
KEITARO_SOURCE=
```

**Важно:** Поля `KEITARO_DOMAIN`, `KEITARO_GROUP` и `KEITARO_SOURCE` являются опциональными. 
Если они не указаны, вы сможете указать их при создании кампании через форму, или оставить пустыми.

**Где взять API ключ:**
1. Войдите в админку Keitaro
2. Перейдите в "Профиль" → "API ключи"
3. Скопируйте существующий ключ или создайте новый
4. Скопируйте также ссылку на документацию API

## Шаг 3: Применение миграций

```bash
python manage.py migrate
```

## Шаг 4: Запуск сервера

```bash
python manage.py runserver
```

Откройте браузер и перейдите на `http://127.0.0.1:8000/`

## Использование

### Создание кампании

1. Нажмите "Создать кампанию" в меню
2. Заполните форму:
   - **Название**: любое имя (например, "Test Campaign")
   - **Geo**: код страны (например, "AU" для Австралии)
   - **Offer ID**: ID оффера из Keitaro
   - **Домен, Группа, Источник**: опциональные поля (можно оставить пустыми)
3. Нажмите "Создать"

Система автоматически:
- Создаст кампанию в Keitaro
- Создаст Flow 1 (фильтр по стране → Google)
- Создаст Flow 2 (редирект на оффер)

### Редактирование кампании

1. Откройте кампанию из списка
2. Нажмите "Редактировать"
3. Добавьте офферы через форму слева
4. Удалите офферы кнопкой "Удалить" в таблице

## Примечания

- Убедитесь, что API ключ имеет права на создание/редактирование кампаний
- Поля "Домен", "Группа" и "Источник" опциональны - можно указать при создании кампании или оставить пустыми
- При ошибках проверьте логи в консоли Django
